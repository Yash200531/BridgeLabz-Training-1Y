# stream-api module

This module demonstrates Java 8 Stream API operations with practical examples.

## Problems implemented

1. **TopTrendingMovies**
   - Filters movies by rating threshold
   - Sorts by rating and year in descending order
   - Limits result to top 5 movies

2. **DoctorAvailability**
   - Filters doctors available on weekends
   - Sorts doctors by specialty and name

3. **ClaimAnalysis**
   - Groups insurance claims by claim type
   - Calculates average claim amount per type

4. **ExpiringMemberships**
   - Filters members whose memberships expire within next 30 days
   - Sorts results by expiry date

5. **TransformNames**
   - Converts names to uppercase
   - Sorts names alphabetically

## Run
Compile and run any class with its `main()` method.
