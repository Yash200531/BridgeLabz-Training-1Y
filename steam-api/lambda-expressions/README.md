# lambda-expressions module

This module demonstrates Java 8 Lambda Expressions, functional interfaces, predicates, comparators, and method references.

## Problems implemented

1. **SmartLighting**
   - Uses a custom functional interface
   - Triggers smart light actions through lambda expressions

2. **ProductSorting**
   - Sorts products using Comparator lambda expressions

3. **NotificationFilter**
   - Filters notifications using Predicate

4. **PatientIDPrinter**
   - Prints patient IDs using method reference

5. **NameUppercase**
   - Uses `String::toUpperCase` for transformation

6. **InvoiceCreator**
   - Uses constructor reference to create invoice objects

## Run
Compile and run any class with its `main()` method.
